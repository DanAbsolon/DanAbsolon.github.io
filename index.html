MERN Stack App: Office Sign In

This app is to be used by visitors to, and staff at, i0 Academy.

On entry, visitors can send their name, company, and sign in time to the database. When exiting the building, they can select themselves from a name search and sign out.

On an admin page, secured by back-end authentication, authorised users can view and sign out (individually or in bulk) all signed in visitors, and view the history of all previous visitors.

In this project, I worked on the back-end: database connection, middleware, and end-points.

JavaScript Game: VelociType

This app fetches randomised text from an API to create a typing speed game.

On keydown, a 60 second countdown starts. Each word typed (including punctuation and capitalisation) is checked on spacebar. If correct, the word becomes green in the source text; red if incorrect.

After 60 seconds, the user gets their speed and accuracy results.

We also created functionality for audio-only gameplay with the Chrome Screen Reader.

React App: Food Delivery

This React app, based on a supplied Bootstrap design, consumes an existing API to provide the user with information on available restaurants.

If the user clicks on a restaraunt, they can view its menu.

The name, carlories and price of each item on the menu are displayed. Some items provide additional information, such as type, depending on the API.

PHP/SQL Collector App: a Bookshelf

In this solo project, isolated values in the database are woven into cohesive sentences displayed on the front-end.

For example, a PHP function checks the last character in the author's name. If this is "s", the possessive marker used in the HTML is " ' ", otherwise the possessive marker is "'s".

Any invalid data causes exceptions to be thrown, caught and their messages logged.
