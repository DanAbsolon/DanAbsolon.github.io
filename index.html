Javascript Typing Game

This Javascript game fetches randomised text from an API to create a typing speed game. On keypress, a 60 second countdown starts. The user types as many words from the text as they can. After 60 seconds, the user gets their speed and accuracy results. We also created functionality for audio-only gameplay with the Chrome Screen Reader.
